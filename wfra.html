Warhammer Fantasy Roleplay Application.

The aim of this project is to create an application that can support players in Warhammer Fantasy Roleplay game. The program will allow users to save their characters, equipment and statistics. It will also have a module for creating a new character. User will be able to modify his or hers characters to reflect the changes during the game. Application will have registration and login functionality.

Self paced project, work in progress.

Project Results

  • The project was developed using the Django framework.
  • The application has been deployed on the PythonAnywhere platform.
  • It features registration and login functionality.
  • Users can create, update, and delete their own characters. Only characters belonging to the logged-in user are visible.
  • Future updates, including equipment management and automation, are planned.
